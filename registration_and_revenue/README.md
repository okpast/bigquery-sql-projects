# Registration and Revenue Analysis

SQL project built using Google BigQuery.

## Description

Analyzes registrations, revenue, and email metrics by country and date.

## Metrics

- Sessions
- Registrations
- Registration Rate
- Revenue
- Revenue by Device
- Revenue by Operating System
- Emails Sent
