# Registration and Revenue Analysis

SQL project created in Google BigQuery.

## Description

Analyzes registrations, revenue, and email metrics by country and date.

## Metrics

- Sessions
- Registrations
- Registration rate
- Revenue
- Revenue by device
- Revenue by operating system
- Emails sent
