# Sent Email Metrics Analysis

SQL project created in Google BigQuery.

## Description

Analyzes marketing performance and email campaign metrics by month.

## Metrics

- Revenue
- Marketing Cost
- Emails Sent
- Email Open Rate
- Email Click Rate
- Registration Count
